# 🧮 문제: 수 나누기 (Dividing the numbers)

- 사이트: 프로그래머스
- 레벨: Lv.00
- 언어: Java

---

## 📌 문제 설명

입력된 정수를 2자리씩 끊어서 나눈 후, 이 값을 더하는 문제.

---

## 💡 풀이 아이디어

- 주어진 수를 100으로 나누어 나머지는 뒤에서 두 자리 수입니다.
- 이 수를 `answer`에 더하고, 다시 `100`으로 나누어 다음 두자리를 넘어갑니다.
- 이 과정을 `number`가 0이 될 때까지 반복합니다.

---

## ✅ 코드

```java
입력 : 4859
while( number > 0){
answer += number % 100;
number /= 100;
}
```
